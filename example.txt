THIS PROGRAM IS A SIMPLE FILE ENCRYPTION AND DECRYPTION TOOL WRITTEN IN PYTHON.

IT WORKS BY TAKING A FILE AND A PASSWORD, THEN CONVERTING THE FILE INTO UNREADABLE DATA USING AES ENCRYPTION. WHEN YOU ENCRYPT A FILE, IT CREATES A NEW FILE WITH A .enc EXTENSION.

TO GET THE ORIGINAL FILE BACK, YOU USE THE SAME PASSWORD. THE PROGRAM READS THE ENCRYPTED FILE, DECRYPTS IT, AND RESTORES THE ORIGINAL CONTENT.

IT ALSO USES A RANDOM SALT AND IV TO MAKE THE ENCRYPTION MORE SECURE, SO EVEN THE SAME FILE WILL PRODUCE DIFFERENT ENCRYPTED OUTPUT EACH TIME.

IN SHORT, IT IS A BASIC TOOL TO PROTECT FILES BY LOCKING AND UNLOCKING THEM USING A PASSWORD.
